🔑 Kredencialet e Supabase
📋 Si të marrësh URL dhe Key nga Supabase
1. Shko te supabase.com → krijo llogari falas → New Project
2. Prit ~2 minuta që projekti të krijohet
3. Shko te: Project Settings → API
4. Kopjo Project URL dhe anon / public key
5. Ngjiti ato në fushat lart dhe kliko Ruaj & Lidhu
🗄️ SQL Setup — Krijo tabelat (ekzekuto 1 herë)
Shko te Supabase → SQL Editor → New Query → ngjit SQL-in poshtë → kliko Run
👑 Bëhu Admin — SQL i fundit
Pasi të regjistrohesh në faqe, ekzekuto këtë në SQL Editor me email-in tënd:
